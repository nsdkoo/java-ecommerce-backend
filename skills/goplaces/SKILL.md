# Google Places 搜索

```
python3 goplaces_wrapper.py search "咖啡店 北京"
python3 goplaces_wrapper.py details ChIJxxx
```

## 配置

- 安装：`brew install steipete/tap/goplaces`
- 环境变量：`GOOGLE_PLACES_API_KEY`

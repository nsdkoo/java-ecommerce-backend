# Keno Winning Numbers Skill

This skill retrieves the latest 10 Keno winning numbers from PlayNow.com.

## Capabilities
- Get the latest 10 Keno draws
- Returns Draw Date, Draw Number, and Winning Numbers

## Usage
When the user asks "Get Keno winning numbers", "Show me the latest Keno results", or "What are the recent Keno draws?", this skill should be invoked.

## Configuration
No specific configuration required. Uses public API.
